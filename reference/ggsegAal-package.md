# ggsegAAL: AAL Atlases for the 'ggseg' Ecosystem

AAL (Automated Anatomical Labeling) atlases for the 'ggseg' ecosystem.
Provides unified 'ggseg_atlas' objects for AAL2 (cortical, Rolls et al.
2015) and AAL3 (cortical, subcortical, cerebellum, Rolls et al. 2020),
with both 2D polygon geometry and 3D vertex/mesh data for use with
'ggseg' and 'ggseg3d'.
